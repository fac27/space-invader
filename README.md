# space-invader

## Cloning this repo
```terminal
git clone https://github.com/fac27/space-invader
npm install
npm run dev
```

You can also visit the deployed site [here](https://subtle-sprite-9d4c0f.netlify.app/)

There are no environment variables needed to run this code base.
